Final Project

For my final project, I will be doing a Macro pad.


Inspirations and references

Jan Lunge - Rotary encoder macro pad - Build & Get most out of it Youtube Video Ryan Bates - Arduino Pro Micro Macro Keyboard Version 2.0 Youtube Video TinyBoatProductions - Programmable Macropad V2 Instructables

3D printing

This is the 3D model that I designed for the macro pad. Do note that the height of the holes made for the rotary encoder is incorrect. there is a need to shorten it.

If you want to design one yourself, some dimensions to take note of is the width of the keys and the diameter of the rotary encoder.


Materials

  • Diodes x 17 (1N4148)
  • Rotary Encoders x2 (EC11)
  • Switches x 20 (Cherry MX Black Hyperglide)
  • Arduino Pro Micro x 1
  • Charging Cable for Arduino x 1

Wiring

Unfortunately, I have wired the switches wrongly, so only some keys are working. I will instead show other's wiring digrams.

Since the arduino has only 16 input pins, for it to register all the keys we need, we can use a combination of their Keypad library and Switch Matrix to register up to a max of 72 keys on an 8x8 matrix.
