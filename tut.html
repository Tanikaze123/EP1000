Let's get right into it:

Fusion 360 .gltf exporter extension

Fusion 360 can not export .gltf files by default, so we have to do it ourselves.

Thankfully, someone on the internet has already done all the hard work for us, so we just need to use an extension.

GLTF Exporter for Autodesk® Fusion 360™ - Trial

Click here to download the trial version of the extension. There are some limitations to it, but it has all that we need. Click on "Fazer download", sign in to Autodesk®, then download and run it

Open Fusion 360

You may need to restart Fusion 360 to see the extension on the top bar. Click on it to reveal the GLTF button. Click on that to export it.


A popup should appear. Save it in your desired folder:



Coding

Time for the boring part. However, this is necessary to get you 3D model up onto the site, so bear with me. I'll give you the codes I used, and you can also check the repository for it too.


Model-Viewer Module

This is the almighty javascript module that allows you to display your 3D model. Don't ask how it works, cause I too, don't know. Check it out here


Import Module

Insert <script type="module" src="https://unpkg.com/@google/model-viewer/dist/model-viewer.min.js"></script> between your <head> tags. Below is how it should look like:


Display

Now we can display the model. Here's my code for it:

<div id="modelBox">
<model-viewer id="c" src="FileName.gltf" camera-controls></model-viewer>
</div>


Extra CSS

Gotta make it look nice right? Here's my CSS codes

#c {
    margin-left: auto;
    margin-right: auto;
    width: 50%;
    display: block;
}

#modelBox {
    width: 100%;
    height: auto;
}

Don't forget to import your CSS files!
